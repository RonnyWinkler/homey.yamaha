...and control your Yamaha Receiver or MusicCast device from Homey.

You can control main functions like:
- on/off
- set input channel
- set sound program
- set volume / mute

Dependent on the selected input you can control your media (Net, USB, CD):
- play / pause
- shuffle / repeat

Media information is shown in the Homey device like artist and track.

To switch on your Receiver, the Network standby must be activated.
